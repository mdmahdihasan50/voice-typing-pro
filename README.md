# Voice Typing Pro

A Windows desktop voice-typing application built with Python. It converts speech to text and can type recognized Bengali or English text into the active application.

## Features

- Voice-to-text recognition
- Bengali and English typing workflows
- Desktop GUI with microphone controls
- Global input simulation
- PyInstaller build configuration

## Setup

```bash
pip install -r requirements.txt
python main.py
```

Generated `build`, `dist` and Python cache files are intentionally excluded from Git.
